# PiNOte
A simple Python Text Editor
Text-Editor
Python Text Editor using Tkinter with tab functionality.


# Menus & Functionality:
File:
New
Open
Save As...
Close
Exit
# Edit and Right click in text window:
Undo
Cut
Copy
Paste
Delete
Select All
# Format:
Word Wrap
Tabs:
# Right Click
New Tab Option

# Keyboard Shortcuts:
Ctrl-s: Save
Ctrl-o: Open
Ctrl-w: Close tab
Ctrl-n: New tab
Ctrl-tab: Next tab
Ctrl-shift-tab: Previous tab
Ctrl-u: Undo
Ctrl-y: Redo
